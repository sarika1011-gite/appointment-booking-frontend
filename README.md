# EditFlow - Appointment Slot Booking & Conflict Prevention System

## Project Overview

EditFlow is a MERN Stack project developed during the Kumarinfotech MERN Stack Internship.

## Technologies Used

- React (Vite)
- Tailwind CSS
- Axios
- React Router DOM
- Node.js
- Express.js
- MongoDB Atlas

## Features

- User Registration & Login
- Appointment Booking
- Appointment Conflict Prevention
- Video Management
- Admin Dashboard
- Responsive Design
